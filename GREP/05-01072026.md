# 05-01072026
Here’s **`grep` file & directory selection control** — how you decide **which files are searched and which are skipped**.

---

## 1. Recursive directory search

### Search current directory recursively
```bash
grep -r "token" .
```

### Recursive, follow symlinks
```bash
grep -R "token" .
```

---

## 2. Include only certain files

### Single extension
```bash
grep -r "token" . --include="*.py"
```

### Multiple extensions
```bash
grep -r "token" . --include="*.py" --include="*.js"
```

---

## 3. Exclude files

### By extension
```bash
grep -r "token" . --exclude="*.min.js"
```

### By filename
```bash
grep -r "token" . --exclude="config.local"
```

---

## 4. Exclude directories

### Single directory
```bash
grep -r "token" . --exclude-dir=node_modules
```

### Multiple directories
```bash
grep -r "token" . --exclude-dir={.git,node_modules,dist}
```

---

## 5. Ignore binary files

### Skip binary files (recommended)
```bash
grep -r -I "token" .
```

### Treat binary as text
```bash
grep -r -a "token" .
```

---

## 6. Limit depth (GNU grep doesn’t support directly)

### Use `find` instead
```bash
find . -maxdepth 2 -type f -name "*.py" -exec grep "token" {} +
```

---

## 7. Follow / ignore symlinks

- `-r` → **don’t** follow symlinks
- `-R` → **do** follow symlinks

---

## 8. Search specific files or directories

### Explicit files
```bash
grep "token" app.py config.py
```

### Explicit directories
```bash
grep -r "token" src/ tests/
```

---

## 9. Hidden files & directories

Included by default:
```bash
grep -r "token" .
```

Exclude explicitly:
```bash
grep -r "token" . --exclude-dir=".*"
```

---

## 10. File lists (advanced control)

### Read files from a list
```bash
grep "token" $(cat files.txt)
```

### Safer (spaces in filenames)
```bash
xargs -0 grep "token" < files.list
```

---

## 11. Combine include + exclude (common pattern)

```bash
grep -r -n "API_KEY" . \
  --include="*.py" \
  --exclude-dir={.git,venv,node_modules}
```

---

## 12. Speed-focused selection

### Search fewer files = faster
```bash
grep -r "token" src --include="*.py"
```

### Skip large dirs
```bash
grep -r "token" . --exclude-dir={.git,node_modules,build}
```

---

## Quick selection reference
```
Recursive search       → -r / -R
Include files          → --include=GLOB
Exclude files          → --exclude=GLOB
Exclude directories    → --exclude-dir=GLOB
Ignore binaries        → -I
Follow symlinks        → -R
Limit depth            → find + grep
```
